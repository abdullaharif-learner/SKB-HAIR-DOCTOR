## ✨ Key Features

* **Real-Time Shop Status Engine:** Instant reactive updates for "SHOP OPEN" and "SHOP CLOSED" states without manual page refreshes.
* **Intelligent Services Marketplace Grid:** Fully dynamic layout rendering prices (in PKR) and remote image assets uploaded directly via the admin panel.
* **Integrated Micro-Interaction Booking Engine:** Seamless appointment scheduling via an interactive, lightweight asynchronous bottom sheet.
* **Live Profile & Modular Navigation Drawer:** Real-time data stream fetching active client credentials, personalized options, and dynamic custom avatars.
* **Session Token Sanitization:** Secure authentication lifecycle ensuring all local session caches and hashes are completely cleared on logout.
